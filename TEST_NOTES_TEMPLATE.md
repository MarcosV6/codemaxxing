## Things to watch for

### 1) Onboarding / first-run
- Was startup clear?
- Any dead ends?
- Did the next step feel obvious?

### 2) Auth flow
- Did `/login` work smoothly?
- Did the model picker open when expected?
- Any weirdness with OpenAI / ChatGPT / Anthropic / Copilot auth?

### 3) Model selection
- Did recommendations make sense?
- Did 14B / 16B feel like the right defaults?
- Was anything misleading?

### 4) Core coding loop
- Did it make good edits?
- Any context misses?
- Any tool weirdness?
- Did approval flow feel smooth or annoying?

### 5) Terminal UX
- Paste behavior
- Escape-to-cancel
- Slash commands
- Picker navigation
- Scrolling / rendering / layout

### 6) Reliability
- Session resume
- Model switching
- Reconnect behavior
- Auth staying alive
- Crashes / freezes / weird state

### 7) Windows-specific issues
- Install friction
- CMD / PowerShell / Windows Terminal differences
- Ollama setup weirdness
- Rendering/input issues
